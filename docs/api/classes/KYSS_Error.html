\KYSS_Error

KYSS Error Class.

Container for checking for KYSS errors and error messages. Return KYSS_Error and use {@link is_kyss_error()} to check if this class is returned. Many KYSS functions pass this class in the event of an error and if not handled properly will result in code errors.
