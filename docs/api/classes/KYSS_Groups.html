add_group()

add_group(string $group, string $display_name, array $permissions) : \KYSS_Group|null

Add group name with permissions to list.

Updates the list of groups, if the group doesn't already exist.

The permissions are defined in the following format array( 'read' => true ); To explicitly deny a group a permission, set the value for that permission to false.

Parameters

string $group

Group name.
string $display_name

Group display name.
array $permissions

List of group permissions in the above format.

Returns

\KYSS_Group|null —

KYSS_Group object if group is added, null if already exists.
