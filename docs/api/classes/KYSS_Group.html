has_permission()

has_permission(string $perm) : bool

Whether group has permission.

The permission is passed through the 'group_has_permission' hook. The first parameter for the hook is the list of permissions the group has granted. The second parameter is the permission name to look. The third and final parameter for the hook is the group slug.

Parameters

string $perm

Permission name.

Returns

bool —

True if group has permission, false otherwise.
