kyss_die()

kyss_die(string $message, string $title, string|array $args)

Kill KYSS execution and display HTML with error message.

This function complements the die() PHP function. The difference is that HTML will be displayed to the user. It is recommended to use this function only when the execution should nnot continue any further. It is not recommended to call this function very often and try to handle as may errors as possible silently.

Parameters

string $message

Error message.
string $title

Error title.
string|array $args

Optional. Arguments to control behavior.

debug_backtrace_summary()

debug_backtrace_summary(string $ignore_class, int $skip_frames, bool $pretty) : string|array

Get functions that have been called until now.

Returns a comma separated string of functions that have been called to get to the current point in the code.

Parameters

string $ignore_class

A class to ignore all function calls within -

                         useful when you want to just give info about the
                         callee.
int $skip_frames

A number of stack frames to skip - useful for unwinding

                     back to the source of the issue.
bool $pretty

Whether or not you want a comma separated string or raw array returned.

Returns

string|array —

Either a string containing a reversed comma separated trace or

                         an array of individual calls.
