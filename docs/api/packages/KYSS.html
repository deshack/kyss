Kill KYSS execution and display HTML with error message.

This function complements the die() PHP function. The difference is that HTML will be displayed to the user. It is not recommended to call this function very often and try to handle as may errors as possible silently.

_deep_replace¶

Perform a deep string replace operation to ensure the values in $search are no longer present.

Repeats the replacement operation until it no longer replaces anything so as to remove "nested" values. E.g. `$needle = '%0%0%0DDD', $haystack = '%0D', $result = ''` rather than the '%0%0DD' that `str_replace()` would return.

trailingslashit¶

Appends a trailing slash.

Will remove trailing forward and backslashes if it exists already before adding a trailing forward slash. This prevents double slashing a string or path. The primary use of this is for paths and thus should be used for paths. It is not restricted to paths and offers no specific path support.
